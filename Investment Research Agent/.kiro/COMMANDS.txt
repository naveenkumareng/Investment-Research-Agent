================================================================================
TROUBLESHOOTING COMMANDS
================================================================================

HARD REFRESH BROWSER (clears cache):
  Windows: Ctrl+Shift+R
  Mac: Cmd+Shift+R

CHECK BROWSER CONSOLE FOR ERRORS:
  Windows: F12
  Mac: Cmd+Option+I
  Look for red error messages in Console tab

CHECK NETWORK REQUESTS:
  F12 → Network tab → Look for red/failed requests

CLEAR BROWSER CACHE:
  F12 → Application → Storage → Clear Site Data

RESTART EVERYTHING:
  1. Stop Terminal 1 (Ctrl+C)
  2. Stop Terminal 2 (Ctrl+C)
  3. Close browser tabs
  4. Start Terminal 1: cd springboot-backend && mvn spring-boot:run
  5. Start Terminal 2: cd "Investment Research Agent" && npm run dev
  6. Open http://localhost:5173/

KILL PROCESS ON PORT (if stuck):
  Windows 8080:
    netstat -ano | findstr :8080
    taskkill /PID <PID> /F
  
  Windows 5173:
    netstat -ano | findstr :5173
    taskkill /PID <PID> /F

================================================================================
API ENDPOINTS (Backend)
================================================================================

Authentication:
  POST   /api/auth/register        - Create new account
  POST   /api/auth/login           - Login with email/password
  POST   /api/auth/refresh         - Refresh access token
  GET    /api/auth/me              - Get current user profile
  GET    /api/auth/health          - Check backend health

Base URL: http://localhost:8080
All requests (except login/register) require: Authorization: Bearer <token>
