REST ENDPOINTS CREATED
════════════════════════════════════════════════════════════════════════════

✅ POST /api/auth/login
   Input:  { email, password, remember }
   Output: { token, refreshToken, user }
   Status: 200 OK or 401 Unauthorized

✅ POST /api/auth/register
   Input:  { name, email, password, confirmPassword }
   Output: { token, refreshToken, user }
   Status: 201 Created or 400 Bad Request

✅ POST /api/auth/refresh
   Input:  { refreshToken }
   Output: { token, refreshToken, user }
   Status: 200 OK or 401 Unauthorized

✅ GET /api/auth/me
   Header: Authorization: Bearer <token>
   Header: X-User-Id: <user-id>
   Output: { id, name, email, role, createdAt }
   Status: 200 OK or 404 Not Found

✅ GET /api/auth/health
   Output: "Auth service is running"
   Status: 200 OK

HOW TO USE
════════════════════════════════════════════════════════════════════════════

STEP 1: START BACKEND
─────────────────────
cd springboot-backend
mvn spring-boot:run

Expected:
  • No errors in console
  • "Tomcat started on port(s): 8080"
  • "Started InvestaApplication"

STEP 2: START FRONTEND (NEW TERMINAL)
───────────────────────────────────────
cd "Investment Research Agent"
npm run dev

Expected:
  • No build errors
  • "VITE v5.0.0 ready"
  • "Local: http://localhost:5173"

STEP 3: REGISTER NEW USER
────────────────────────
1. Go to http://localhost:5173/auth/register
2. Fill form:
   • Name: Your Name
   • Email: test@example.com
   • Password: SecurePass123
   • Confirm: SecurePass123
3. Click Register
4. Should auto-login and redirect to dashboard

STEP 4: VERIFY
──────────────
DevTools (F12):
  • Application → Local Storage
  • Check: auth.token, auth.refresh, auth.user
  • Network Tab
  • Check: Authorization header on requests

MongoDB Atlas:
  • Check: users collection has your user
  • Check: password is hashed ($2a$12...)
