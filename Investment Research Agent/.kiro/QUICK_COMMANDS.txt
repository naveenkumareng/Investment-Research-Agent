TROUBLESHOOTING COMMANDS
────────────────────────────────────────────────────────────────────────────

Clear cache and rebuild:
    cd "Investment Research Agent"
    rm -rf node_modules dist .next
    npm install
    npm run build
    npm run dev

Kill process on port 5173 (frontend):
    # Find process
    netstat -ano | findstr :5173
    # Kill by PID (replace XXXX with PID)
    taskkill /PID XXXX /F

Kill process on port 8080 (backend):
    # Find process
    netstat -ano | findstr :8080
    # Kill by PID (replace XXXX with PID)
    taskkill /PID XXXX /F

Clean build frontend:
    cd "Investment Research Agent"
    npm run build

Check frontend errors:
    npm run dev
    # Open browser F12 → Console tab
    # Look for red errors

Check backend logs:
    # Look at Maven terminal output
    # Should see "Tomcat started on port(s): 8080"
    # MongoDB connection messages

Reset database (MongoDB):
    # Clear users collection to reset accounts
    # Go to MongoDB Atlas → Collections → Delete all documents
